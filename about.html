Acerca de Nosotros

El equipo compuesto por Kevin Lagos (Cuenta: 12011007), Marlon Rubí (Cuenta: 11641115), Alejandro Romero (Cuenta: 11941338), Heriberto Rueda (Cuenta: 11811150), Maria Fernanda Maldonado (Cuenta: 11841346) y Maria Celeste Lupy (Cuenta: 11941104) se unió para crear una aplicación web como parte de su curso de Diseño Web 2. El objetivo del proyecto era desarrollar una aplicación web que combinara creatividad y funcionalidad para ofrecer a los usuarios una experiencia única en línea. Durante el proceso de desarrollo, el equipo se esforzó por aplicar los principios de diseño web modernos y las mejores prácticas de usabilidad.
